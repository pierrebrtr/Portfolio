Moov'in Nantes

An Android app used in a French city (Nantes) to know the bus and tram timetables, calculate a route but also to see the available parking spaces or the bicycles available in the city.

About the project.

2016-2018
non-profit - Education purpose

This project was designed to learn JAVA at different levels of complexity. I was able to work on many different technologies while relying on a REST API proposed by the city of Nantes.

Features: Route calculation, bus and tram timetables, availability of parking and self-service bicycles, bus ticket prices, management of favourite stops, notifications and alerts, theme management, news and weather.

Purpose

This app was created with the aim of learning by myself.

Language

This app was developed in java.

Deployment

The application has been downloaded more than 600 times.
